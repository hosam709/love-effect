
# Love Effect Page

ملف بسيط لعرض كلمات جميلة مع تأثير كتابة،
ويعرض نسبة البطارية وبعض معلومات الاتصال (إن كان المتصفح يدعم).

## الاستخدام مع GitHub Pages
- ارفع الملفات في ريبو عام على GitHub.
- من الإعدادات (Settings) -> Pages اختر الفرع `main` والمجلد `root`.
- بعد دقائق هتحصل على لينك شغال للموقع.
